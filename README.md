# Proyecto de Casos Clínicos Interactivos

Este proyecto es una plataforma web educativa diseñada para la visualización y análisis de casos clínicos complejos. El objetivo es proporcionar una experiencia de aprendizaje inmersiva mediante el uso de modelos 3D interactivos e infografías detalladas.

## 🚀 Accede al proyecto
Puedes explorar los casos clínicos publicados en tiempo real a través del siguiente enlace:

👉 **[Ver Casos Clínicos Interactivos](https://camiloalarcon25.github.io/Proyecto-Casos-Cl-nicos/)**

---

## 🛠️ Tecnologías Utilizadas
Este proyecto ha sido desarrollado siguiendo estándares web modernos para garantizar una navegación fluida y compatible con diversos dispositivos:

*   **HTML5 & CSS3:** Estructura semántica y diseño responsivo con técnicas de Flexbox y Grid.
*   **JavaScript:** Lógica para la interactividad de los carruseles y la manipulación del DOM.
*   **Model-viewer:** Librería de Google para la renderización y visualización de modelos 3D (`.glb`) en el navegador.
*   **Fuentes tipográficas:** Implementación de *Playfair Display* e *Inter* vía Google Fonts para un acabado estético profesional.

---

## 📋 Características Principales

*   **Visualización 3D:** Integración de modelos anatómicos detallados que permiten rotación y control de cámara.
*   **Carruseles Informativos:** Sistema de navegación para infografías que permite al usuario analizar patologías de forma secuencial.
*   **Diseño Adaptable (Responsive):** La interfaz se adapta automáticamente tanto a ordenadores de escritorio como a dispositivos móviles, garantizando una experiencia óptima en cualquier tamaño de pantalla.
*   **Optimización de Carga:** Estructura modular que permite la carga independiente de cada caso clínico.
